# Installing

1.  Install [Python](https://www.python.org/)
2.  Clone or download the [OneTMY repository](https://github.com/e-marco/onetmy):
```bash
git clone https://github.com/e-marco/onetmy.git
```
3.  Change directory and install the Python requirements:
```bash
cd onetmy
pip install -r requirements.txt
```
